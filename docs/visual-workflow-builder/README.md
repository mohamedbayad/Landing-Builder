# Visual Workflow Automation Builder

This folder contains implementation-ready planning artifacts derived from the product PRD.

## Contents

1. `01-epics-sprints.md`: 12-week plan with epics, stories, acceptance criteria, and sprint sequencing.
2. `02-data-model-events.md`: PostgreSQL schema direction, execution/state model, and queue/event contracts.
3. `03-api-v1.yaml`: OpenAPI 3.1 draft for MVP endpoints.
4. `04-frontend-ia-component-map.md`: React Flow information architecture and component ownership map.

## MVP Scope Snapshot

- Channels in V1: `email`, `whatsapp`
- Core blocks in V1: Trigger, Send Message, Send Sequence, Delay, Branch, Add/Remove Tag, Goal, End Workflow
- Out of V1: A/B testing, templates marketplace, collaboration, multi-workspace, Instagram/SMS execution
