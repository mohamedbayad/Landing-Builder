# Ad Creative Strategy

Plan performance visuals that stop scroll, build trust, and increase perceived value.
Balance aggressive ad hooks with premium brand presentation.

## Required Inputs

1. Product Identity Profile (`product-research`).
2. Conversion blueprint (`landing-page-conversion-engine`).
3. Product Identity Lock (`product-image-safety`).

## Angle Prioritization

Score angles on:
1. Emotional tension.
2. Urgency.
3. Clarity in 1 second.
4. Visual distinctiveness.

Select top 3 angles with mandatory coverage:
1. Emergency angle.
2. Convenience angle.
3. Lifestyle/identity angle.

## Mandatory Creative Pack

Plan at least these visuals:
1. Emergency Scene:
   - high-stakes context,
   - visible tension,
   - product as immediate tool.
2. Lifestyle Usage:
   - aspirational real-life context,
   - premium but believable.
3. Close-Up Action Shot:
   - hands interacting with mechanism,
   - result visibly in progress.
4. Before/After Split:
   - pain state vs outcome state,
   - strong visual contrast.
5. Trust/Proof Visual:
   - UGC or review-led context.

## Scroll-Stop Logic

Each creative must define:
1. First frame / first glance.
2. Pattern interrupt element.
3. Stakes cue ("what goes wrong").
4. Resolution cue ("what changes").

If first-glance message is unclear, redesign.

## Variation System

Create two variants per concept:
1. `A_premium_clean`: refined, controlled, high-value.
2. `B_raw_aggressive`: urgent, emotional, high tension.

At least one `B_raw_aggressive` must be used in the ad set.

## Validation Gate

Reject if:
- Feels like stock photography.
- No visible stakes or emotion.
- Product identity mismatches lock.
- Hand interaction or physics looks fake.

Accept only if:
- Message is instantly clear.
- Emotion is immediate.
- Product looks real and correctly integrated.
- Visual supports both conversion and perceived value.

## Output Format

Return:
1. Angle score table.
2. Creative pack plan (5 mandatory visuals).
3. Hook logic for each visual.
4. A/B variant directives.
5. Validation scorecard.
