# SEO Content Optimizer

Single workflow for the full content lifecycle: Plan → Write → Audit → Refresh. Creates comprehensive, engaging content optimized for both search engines and users.

---

## When to use this skill

- When planning a content strategy or topic cluster
- When writing SEO-optimized articles, landing pages, or blog posts
- When auditing existing content for quality, E-E-A-T signals, and keyword optimization
- When refreshing outdated content to maintain rankings

## How to use it

### Phase 1 — Plan

1. Analyze the main topic comprehensively
2. Map search intent variations (informational, commercial, transactional)
3. Identify subtopics and content gaps
4. Create a detailed outline with H1/H2/H3 structure
5. Set target word count and primary/secondary keywords
6. Plan internal linking strategy

**Topic Cluster Components:**
- Pillar page (comprehensive guide)
- Supporting articles (subtopics)
- FAQ and glossary content
- Comparison/versus content

### Phase 2 — Write

**Introduction (50-100 words):** Hook + value proposition + primary keyword.

**Body:** Comprehensive coverage, logical flow, supporting data, natural keyword placement (0.5-1.5% density), semantic variations, clear H2/H3 subheadings.

**Conclusion:** Summarize key points, clear CTA, reinforce value.

**Quality standards:** Original content, grade 8-10 reading level, short paragraphs (2-3 sentences), bullet points for scannability.

### Phase 3 — Audit

Score existing content on:
| Category | What to Check |
|---|---|
| Content Depth | Missing subtopics, thin sections |
| E-E-A-T Signals | Author bio, credentials, citations |
| Readability | Paragraph length, clarity, structure |
| Keyword Optimization | Density, placement, semantic variations |
| Trust Indicators | Data, statistics, expert quotes |

### Phase 4 — Refresh

Check for:
- Statistics older than 2 years
- Dated examples or case studies
- Missing recent industry developments
- Expired links or references

**Refresh priority:** Pages losing rankings > High-traffic pages declining > Stagnant pages (6+ months).

## Examples

### Example 1 — Content Outline

**Input:** "Best portable tire inflator 2026"  
**Output:**
```
Title: Best Portable Tire Inflator 2026
Intent: Commercial
Word Count: 2500

I. Introduction — Why every driver needs one
II. Top 5 Picks (with specs comparison table)
III. How to Choose (PSI, power source, size)
IV. Common Mistakes When Buying
V. FAQ
```

### Example 2 — Content Audit Finding

**Input:** Existing blog post on tire inflators  
**Output:**
```
Content Depth: 6/10 — Missing comparison table and FAQ section
E-E-A-T: 4/10 — No author bio, no citations
Recommendation: Add author credentials, include 3 data sources, add FAQ schema
```

---

## Constraints

- **Do NOT** publish unreviewed AI-generated content — always verify accuracy
- **Do NOT** keyword stuff — 0.5-1.5% density maximum
- **Do NOT** write generic filler — every paragraph must deliver value
- **Do NOT** skip E-E-A-T signals — include author info, data sources, expert quotes
- **Do NOT** refresh content by only changing dates — add substantive new information
- **ALWAYS** include a primary keyword in the first 100 words naturally
- **ALWAYS** write for humans first, optimize for search engines second
