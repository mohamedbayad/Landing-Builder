# Landing Page Section Library

Use a narrative conversion flow that builds tension, proves value, and closes with confidence.
Use `landing-page-design` for visual system and `landing-page-social-proof` for proof content blocks.

## Core Flow

### Universal Premium Flow
1. Hero
2. Rating Summary / Trust Strip
3. Problem Diagnosis
4. Problem Agitation Story (mandatory for problem-solving products)
5. Demo / How It Works (mandatory)
6. Solution Reveal
7. Features with benefit translation
8. Use-Case Visualization
9. Social Proof Cluster
10. Guarantee / Risk Reversal
11. Offer Stack + CTA
12. FAQ
13. Footer CTA

Do not jump from Hero to Solution.

## Mandatory Demo Section

Include a dedicated demonstration section for all functional/problem-solving products.

### Demo Section Requirements
1. Step-by-step mechanism (3 steps max).
2. Clear input -> action -> result logic.
3. Visual explanation (sequence images or video-style frame layout).
4. One proof point tied to the mechanism ("why this works").

Preferred micro-flow:
1. `Step 1`: Trigger/use start.
2. `Step 2`: Product action ("push button -> process").
3. `Step 3`: Outcome/result.

## High-Impact Section Rules

### Problem Agitation Story
Must include:
1. Real-world trigger moment.
2. Escalation/tension.
3. Cost of delay.

### Use-Case Visualization
Include 3 contexts:
1. Emergency/high-stress.
2. Daily convenience.
3. Long-term prevention/confidence.

### Social Proof Cluster
Include:
1. Rating summary near top.
2. Testimonial grid before offer.
3. Final proof reminder before last CTA.

## Layout and UX Constraints

- No navigation menu.
- No external links in conversion path.
- One section, one purpose.
- CTA + micro-trust in hero, demo section, offer section, footer.
- Keep feature cards at 4-6 max.

## Non-Negotiable Rules

- Do not skip Demo section for functional products.
- Do not skip Problem Agitation for problem-solving products.
- Do not place all trust/proof in a single block.
- Do not use generic testimonial cards without scenario + emotional result.
- Always preserve clear visual rhythm and premium spacing.
