# SEO Meta Optimizer

Creates compelling, keyword-rich meta titles, descriptions, and URL suggestions that maximize click-through rates while following character limits and best practices.

---

## When to use this skill

- When creating meta titles and descriptions for new content
- When optimizing existing metadata for better CTR
- When generating URL slug suggestions for new pages
- When A/B testing metadata variations

## How to use it

### URL Optimization

- Keep under 60 characters
- Use hyphens, lowercase only
- Include primary keyword early
- Remove stop words when possible

### Title Tag Optimization

- 50-60 characters (pixels vary by device)
- Primary keyword in first 30 characters
- Include emotional triggers or power words
- Add numbers or year for freshness signals
- Brand placement: beginning for brand awareness, end for SEO

### Meta Description Optimization

- 150-160 characters optimal
- Include primary + secondary keywords naturally
- Use action verbs and benefits
- Add compelling CTA
- Special characters for visibility (✓ → ★) used sparingly

### Output Package

Always deliver 3-5 variations for each element:

```
URL: /optimized-slug-here
Title 1: Primary Keyword - Benefit Hook | Brand (55 chars)
Title 2: How to [Action] with [Product] — [Year] Guide (52 chars)
Title 3: [Number] Reasons [Product] Is [Claim] | Brand (48 chars)
Description 1: Action verb + benefit. Clear CTA ✓ (155 chars)
Description 2: [Alternate angle with different emotional trigger] (158 chars)
```

---

## Constraints

- **Do NOT** exceed 60 characters for title tags — truncation kills CTR
- **Do NOT** exceed 160 characters for meta descriptions
- **Do NOT** stuff keywords into metadata — keep it natural and compelling
- **Do NOT** use the same title/description across multiple pages
- **ALWAYS** deliver at least 3 variations for A/B testing
- **ALWAYS** count characters precisely — include spaces and punctuation
